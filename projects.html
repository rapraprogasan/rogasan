Project Description: BuddyBuy – Static E-Commerce Website

Project Name: BuddyBuy

Type: Static Website

Category: E-Commerce

Purpose

This is our first project in Human-Computer Interaction (HCI) during our first year. It is a simple static online shopping website designed to showcase basic web development skills using only HTML, CSS, and JavaScript.

Features

Homepage: A welcoming page with an introduction to BuddyBuy and a navigation menu. Product Listings: Displays featured and latest products with images, descriptions, ratings, and prices. Navigation Menu: Links to Home, Products, About, Contact, and Account pages. Shopping Cart Icon: A simple cart icon linking to a cart page (non-functional, only for display). Testimonials: Customer reviews with star ratings. Brands Section: Logos of partner brands. Footer: Contains social media links, company information, and useful links. Basic JavaScript: Includes a menu toggle script for mobile responsiveness.

Technology Stack: Frontend: HTML, CSS, JavaScript Backend: None (Static website, no database or server-side functionalities) This project serves as a foundational exercise in web development and user experience design, focusing on structuring and styling an e-commerce interface without dynamic functionalities.
